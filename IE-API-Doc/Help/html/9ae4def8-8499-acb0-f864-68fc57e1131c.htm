An API for connecting HBM weighing devices (e.g. WTX or DSE) via ethernet
Classes

BaseWTDevice
Basic device class, holds the most important properties, available in all weighing devices.

- Weight data properties (e.g. Gross/Net value or tare value) and methods (e.g. Tare(), Zero() )

- Adjustment properties (e.g. zero signal or nominal signal) and methods (e.g. CalibrateZero() )
